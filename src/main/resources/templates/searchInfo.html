The Search Formular

You search by selecting a site in which you wish to search, and entering a search query to search for.

Site Selection

There has been added a number of sites to the Dev-Search facility. These sites have been crawled and prepared for searching. You can only search inside a single site at the time.

Search Language

The query language in version {version} of the Dev-Search application is based upon the simple queries of Elastic Search
A query can look like this: "Fault Tolerance" +microprofile. This query searches for pages having the term Fault Tolerance and the word microprofile.
